## Modules structure

`persistent` - Data Access Layer: contains models and repositories.

`logic` - Business Logic Layer: contains services.

`api` - Web API Layer: contains web controllers.

## Dependencies

The work on the project consists of the following

1. Java 1.8
1. Gradle 2.1+
1. PostgreSQL 9.+
1. Wildfly 8.0
